# Hashing

## Overview

This library provides some hashing utilities for libraires in the Roblox Platform.

## Features

This provides mechanisms for hashing data.

It provides implementations for the following hashes:

- Consistent Hashing
- Consistent Hashing (v2)
- HMAC
- MurmurHash3
- SHA256
